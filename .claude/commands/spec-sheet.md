1. What This App Does
SylcRoad is Social Sculpâ€™s all-in-one platform for creator marketing on Instagram and TikTok. It helps the team plan and run campaigns, gives clients an insights dashboard, and lets creators receive assignments and track payments. Brands set a monthly budget that is distributed across creators.

2. Core Features
- Campaign Builder: Create briefs, set goals and monthly budgets, auto-suggest or manually allocate funds across creators, and approve/publish timelines. Custom implementation using Next.js 15 for the UI and Convex functions for budget allocation logic, approvals, and real-time updates.
- Creator Workroom: Assign briefs, collect and review content with compliance checks, schedule posts, manage revisions, and trigger payment milestones after approval. Custom implementation using Clerk-secured workflows in Next.js and Convex for state management, content review logs, and milestone tracking.

3. Tech Stack
- Framework: Next.js 15
- Database/Backend: Convex
- Auth: Clerk
- Social data integrations: Instagram Graph API, TikTok for Developers (metrics and content metadata)
- Feature-specific: Custom budget allocation and compliance engines built on Convex

4. UI Design Style
Modern, clean, intuitive UI without being flashy.
